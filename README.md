# Bike Rental Demand Prediction 

## 📌 Overview

This project performs a regression analysis to investigate how different factors influence the daily number of rented bikes (`cnt`). The goal is to build a predictive model that estimates bike demand and provides insights into the key drivers affecting usage.

---

## 📊 Key Tasks

* Exploratory Data Analysis (EDA)
* Visualization using scatter plots and correlation heatmaps
* Multiple Linear Regression modeling
* Stepwise model selection
* Statistical inference (p-values, confidence intervals)
* Residual diagnostics (linearity, normality, homoskedasticity)
* Multicollinearity analysis using Variance Inflation Factor (VIF)

---

## ⚙️ Methodology

1. **EDA**
   Examined relationships between variables using visualizations and correlation analysis.

2. **Model Building**
   Fitted a multiple linear regression model using all available predictors.

3. **Model Selection**
   Applied stepwise selection to identify the most relevant variables.

4. **Diagnostics**
   Evaluated regression assumptions through residual analysis.

5. **Final Model**
   Selected based on statistical significance, goodness-of-fit, and diagnostic checks.

---

## 📈 Results

* Identified significant factors affecting bike rental demand
* Built an interpretable regression model
* Evaluated model performance using standard metrics (e.g., R²)
